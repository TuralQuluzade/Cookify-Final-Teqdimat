# Cookify

Cookify is a full-stack recipe sharing platform developed as a graduation project at Code Academy. The project was successfully presented and evaluated by the academy's instructors and reviewers, resulting in the award of a Certificate and Diploma of Completion.

The platform allows users to discover, create, and share recipes while interacting with other members through comments and likes. It was built with modern web development technologies and follows industry-standard practices for authentication, API development, and state management.

## Key Features

* User registration and authentication (JWT)
* Recipe creation and management
* Blog system with comments and likes
* RESTful API architecture
* Secure user authorization
* Responsive and user-friendly interface

## Technologies Used

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Redux Toolkit
* React.js
